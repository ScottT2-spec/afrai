# AfrAI — AI Infrastructure for Africa 🌍

Smart model routing, semantic caching, and offline-first AI gateway built for the African continent.

**[View API Documentation →](/docs)**

## Features

- 🧠 **Smart Routing** — Complexity-aware model selection with adaptive ML learning
- 🔄 **Multi-Provider** — OpenAI, Anthropic, Google, Groq, SambaNova with fallback chains
- 💰 **Mobile Money** — First AI API platform to accept MTN MoMo payments 🇬🇭
- ⚡ **Circuit Breaker** — Per-provider health tracking with automatic failover
- 🔒 **Rate Limiting** — Token-aware sliding window
- 📊 **Usage Tracking** — Per-request cost and token billing

## Tech Stack

TypeScript · Fastify · PostgreSQL · Redis · Drizzle ORM · Zod · ONNX Runtime
